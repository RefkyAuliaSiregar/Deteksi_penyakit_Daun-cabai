# penyakit_daun

Penelitian ini berfokus kepada pengembangan aplikasi deteksi penyakit pada daun tanaman cabai dengan model yang telah dibangun menggunakan pendekatan deep learning EfficientDet. Aplikasi deteksi penyakit pada daun tanaman cabai sebanyak lima kelas yaitu daun sehat, bercak daun, daun keriting, virus kuning daun, dan whitefly daun ini dikembangkan menggunakan SDK Flutter agar dapat digunakan secara offline oleh pengguna (petani) menggunakan smartphone. Aplikasi dikembangkan dengan metode agile agar pengembangan aplikasi lebih fleksibel.
